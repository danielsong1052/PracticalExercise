# PracticalExercise
This is the Practical Exercise using Angular 6 and Java based on Spring Boot and Mysql.

## How to run
- Database : run Mysql
- Front-End : ng serve
- Back-End : run "o.s.b.w.embedded.tomcat.TomcatWebServer" on Boot Dashboard
- Link to "https://localhost:4200"
