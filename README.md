# PracticalExercise
This is the Practical Exercise.

## How to run
- Database : run Mysql
- Front-End : ng serve
- Back-End : run "o.s.b.w.embedded.tomcat.TomcatWebServer" on Boot Dashboard
- Link to "https://localhost:4200"

## Technologies used
- Spring Boot
- Hibernate / JPA
- Angular 6
- Bootstrap
- Angular Material Design
- Mysql
